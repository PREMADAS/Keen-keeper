# 🌿 KeenKeeper

A personal relationship management app to help you stay connected with the people who matter most.

## 📝 Description

KeenKeeper is your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most — track when you last reached out, set goals, and never let important friendships fade.

## 🛠️ Technologies Used

- React 19
- React Router
- Tailwind CSS
- DaisyUI
- Recharts
- Vite

## ✨ Key Features

- **Friend Tracking** — View all your friends in one place with their contact status, tags, and last interaction date.
- **Quick Check-In** — Log a Call, Text, or Video interaction with one click and instantly see it reflected in your Timeline.
- **Friendship Analytics** — Visualize your interaction habits with a live donut chart showing Call, Text, and Video breakdowns.
